# AI 学习世界 · 互动地图原型

这是一个零依赖的网页原型，已经串起：

- 世界地图与 5 个课程大陆
- 点击“启程之森”进入第一章子地图
- 7 个依次解锁的课程站点
- 课程信息面板、完成反馈与本地进度保存
- 桌面与手机尺寸适配、键盘焦点和减少动态效果支持

## 运行

直接双击 `index.html` 即可。也可以在上一级“人工智能课程”目录启动静态服务器，例如：

```powershell
cd "F:\人工智能课程"
python -m http.server 8000
```

然后访问 `http://localhost:8000/interactive-map/`。

## 后续内容接入

地图与课程配置集中在 `app.js` 顶部的 `continents` 和 `lessons` 数组中。后续把每一站的正式课件、练习或独立页面挂到对应配置即可。
