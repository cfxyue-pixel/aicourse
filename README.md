# AI 学习世界 · 互动地图原型

这是一个零依赖的网页原型，已经串起：

- 世界地图与 5 个课程大陆
- 点击“启程之森”进入第一章子地图
- 7 个依次解锁的课程站点
- 沉浸式全屏课程场景、完成反馈与本地进度保存
- 独立的“星群信标”K-means 聚类游戏
- 桌面与手机尺寸适配、键盘焦点和减少动态效果支持

## 第一课交互结构

普通站点以场景化页面展示课程内容，不使用统一的居中弹窗。第六站“聚类传送门”拥有独立游戏界面，玩家需要：

1. 判断聚类数量 K；
2. 选择三个初始聚类中心；
3. 按最近距离划分样本；
4. 把中心移动到组内平均位置；
5. 重复划分与更新，直到中心收敛。

游戏完成后自动解锁“课程港口”。

## 运行

直接双击 `index.html` 即可。也可以在上一级“人工智能课程”目录启动静态服务器，例如：

```powershell
cd "F:\人工智能课程"
python -m http.server 8000
```

然后访问 `http://localhost:8000/interactive-map/`。

## 后续内容接入

地图与课程配置集中在 `app.js` 顶部的 `continents` 和 `lessons` 数组中。后续把每一站的正式课件、练习或独立页面挂到对应配置即可。
